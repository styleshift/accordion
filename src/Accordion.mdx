# Accordion

A vertically stacked set of interactive headings that each reveal a section of content.

```tsx
<Accordion>
  <Accordion.Item>
    <Accordion.Trigger>Section 1</Accordion.Trigger>
    <Accordion.Content>Content for section 1</Accordion.Content>
  </Accordion.Item>
</Accordion>
```
