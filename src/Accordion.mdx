# Accordion

A vertically stacked set of interactive headings that each reveal a section of content.

## Usage

The Accordion component is useful for:

- Organizing related content into collapsible sections
- Reducing visual clutter in content-heavy interfaces
- Creating FAQ sections or similar expandable content areas
